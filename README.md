# PiValidator

Abstraction of a validator encapsulation

## Install

```bash
npm install pi-validator
```

## Usage

```javascript
var PiValidator = require('pi-validator');

class CustomValidator extends Validator {
  validate() {
    // Your custom validation
  }
}
```
